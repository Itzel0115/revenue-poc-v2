## 4. 異常偵測抽查

抽查 3 筆異常，人工確認是否為真實異常：

### 異常 1：事業群 1 / 平台 GG-02 / 2024-06，營收/庫存金額比偏低

- **程式訊號值：** 0.7315
- **手算驗算：** 該月 GG-02 營收 790，庫存金額 1,080；790 ÷ 1,080 = 0.7315 ✅
- **合理性判斷：** ✅ 真實異常。平台 GG-02 自 2024-06 起營收下滑但庫存持續升高，比值低於門檻 0.80，符合「庫存積壓、銷售動能減弱」的管理訊號。

### 異常 2：事業群 1 / 平台 GG-02 / 2024-07，營收/庫存金額比偏低

- **程式訊號值：** 0.6281
- **合理性判斷：** ✅ 真實異常。為異常 1 的延續，比值持續惡化（0.73 → 0.63），趨勢一致。

### 異常 3：事業群 3 / 平台 GG-06 / 庫存持續升但營收持續降

- **合理性判斷：** ✅ 真實異常。GG-06 自 2024-02 起每月營收持續下滑（-3% ~ -9%），庫存金額卻持續上升（+1% ~ +6%），方向性背離持續 7 個月，符合典型庫存風險訊號。

**結論：** 3/3 抽查均為真實異常，無誤報。
